# teamOS-kernel
An independent, bug-free x86 hobbyist operating system built from scratch. Features a 2.89MB footprint, custom NASM/C kernel, native 1251 Cyrillic character support, large-number calculator, and a text-mode paint UI. No Linux code. Runs instantly in the browser via v86!
